# ⚡ تذكير: ميزة الإجراءات السريعة (Quick Actions)

## متى تستخدمها؟

لما يجيك تنبيه عن حدث مهم (مثل بوت جديد أو عضو مشبوه)، البوت بيرسل لك **خيارات سريعة** في نهاية التنبيه.

---

## 🎯 كيف تستخدمها؟

### الطريقة 1: رقم فقط (للتنبيه الأخير)
```
1
```
أو
```
2
```

### الطريقة 2: كود الإجراء + رقم (لتنبيه معين)
```
ABC123 1
```

---

## 📋 الخيارات المتاحة

حسب نوع الحدث، بتشوف خيارات مثل:

### 🤖 لما يضاف بوت جديد:
```
1 → Ban Bot (حظر البوت)
2 → Kick Bot (طرد البوت)
3 → Strip Roles (سحب الأدوار)
4 → Get Info (معلومات)
5 → Ignore (تجاهل)
```

### 👤 لما عضو مشبوه يدخل:
```
1 → Ban (حظر)
2 → Timeout (كتم)
3 → Watch (مراقبة)
4 → Get Info (معلومات)
5 → Ignore (تجاهل)
```

### ⚙️ لما أحد يغير أدوار:
```
1 → Strip Roles (سحب الأدوار)
2 → Get Info (معلومات)
3 → Ignore (تجاهل)
```

---

## ⏰ مهم: المدة المتاحة

- عندك **5 دقايق** عشان ترد
- بعد 5 دقايق، الإجراء بينتهي تلقائياً

---

## 💡 مثال عملي

**تنبيه يوصلك:**
```
🔴 CRITICAL BOT ADDED TO SERVER

**Bot:** SuspiciousBot#1234 (123456789)
**Account Age:** ⚠️ 2 days old (RECENT)
**Added by:** @Admin (987654321)
**Risk Level:** 🔴 CRITICAL
**Permissions:** 🔴 ADMINISTRATOR

────────────────────────────────────────
⚡ QUICK ACTIONS [ABC123]
────────────────────────────────────────
**رد برقم الإجراء المطلوب:**

  1 → 🔨 Ban Bot
  2 → 🗑️ Kick Bot
  3 → ⚠️ Strip Roles
  4 → ℹ️ Get Info
  5 → ❌ Ignore

⏰ Expires in 5 minutes
────────────────────────────────────────
```

**أنت ترد:**
```
1
```

**البوت يرد:**
```
✅ 🔨 Ban Bot
✅ Banned user 123456789
```

---

## ⚙️ التحكم بالميزة

لو تبغى تطفي هالميزة، غير في `config.py`:
```python
QUICK_ACTIONS_ENABLED = False
```

أو غير المدة المتاحة:
```python
QUICK_ACTION_TIMEOUT = 300  # بالثواني (300 = 5 دقايق)
```

---

## 🚀 فوائد الميزة

- ⚡ **رد فوري** - ما تحتاج تكتب أوامر طويلة
- 🎯 **دقيق** - تختار الإجراء المناسب بسرعة
- 🔒 **آمن** - الإجراءات تنتهي تلقائياً
- 📊 **مرتب** - كل شيء في تنبيه واحد
